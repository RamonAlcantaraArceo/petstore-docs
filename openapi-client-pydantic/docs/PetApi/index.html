    petstore_openapi_client.PetApi¶

    All URIs are relative to http://localhost:8000

    Method HTTP request Description
    add_pet POST /api/v1/pet Add Pet
    delete_pet DELETE /api/v1/pet/{pet_id} Delete Pet
    find_pets_by_status GET /api/v1/pet/findByStatus Find Pets By Status
    find_pets_by_tags GET /api/v1/pet/findByTags Find Pets By Tags
    get_pet_by_id GET /api/v1/pet/{pet_id} Get Pet By Id
    update_pet PUT /api/v1/pet Update Pet
    update_pet_with_form POST /api/v1/pet/{pet_id} Update Pet With Form
    upload_file POST /api/v1/pet/{pet_id}/uploadFile Upload File

    add_pet¶

    Pet add_pet(pet_create)

    Add Pet

    Add a new pet to the store.

    Args: pet: Pet data from request body. service: Injected PetService.

    Returns: The created pet.

    Example¶

    host = "http://localhost:8000"
